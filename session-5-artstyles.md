# Session 5 — Art Direction & Image Prompts

**Purpose:** Visual assets for *The Flame Ascendant* — handouts, projector slides, VTT tokens, post-session recap. Prompts are Midjourney v7 RAW format. Paste verbatim into the MJ Discord bot.

---

## CORE STYLE LOCK (use across every prompt)

Append this style suffix to every scene prompt so the campaign looks like one campaign, not ten:

```
dark fantasy oil-painting, chiaroscuro lighting, muted blood-red and ash-grey palette with single accent of pale silver-moonlight, painterly brushwork in the style of Zdzisław Beksiński crossed with Gerald Brom, heavy atmospheric perspective, gore-smog haze, no logos no text no UI, --style raw --v 7 --s 250 --ar 16:9
```

**Why these choices:**
- **Beksiński** — corrosion, decay, biological horror without explicit gore (matches Avernus tone without TTRPG cliché)
- **Brom** — figurative anchoring, demonic anatomy reads as *character* not monster
- **`--style raw`** — kills MJ's default romance-glow, preserves grit
- **`--s 250`** — mid-stylize, lets the prompt drive composition not MJ's defaults
- **Color discipline** — every scene is red/grey/black with ONE silver-moonlight accent for the Moonkite / Aurora / divine moments

**Palette swap rules:**
- Default = ash-grey + blood-red + silver-moonlight accent
- Bhaal-heavy scene → swap silver for **bone-white** accent
- Coram-heavy scene (future) → swap red for **honey-gold** accent
- Pure Hell wilderness → swap silver for **infernal-orange** ember accent

**Aspect ratio guide:**
- Handouts (letter, map) → `--ar 3:4` (portrait, fits parchment)
- Read-aloud establishing shots → `--ar 21:9` (cinemascope, projector)
- NPC portraits → `--ar 2:3` (token-ready)
- Tokens for VTT → `--ar 1:1`

## QUICK PRINTING / DISPLAY WORKFLOW

1. **Run all 14 prompts in MJ in a batch.** ~30 minutes total. Pick 1 of 4 from each.
2. **Letter + Map handouts:** Overlay text in Canva or Photoshop, print on cream cardstock at 6×9 inches, crumple once, lightly dab corners with red food coloring + coffee.
3. **Projector slides:** Drop the 21:9 shots into Keynote/PowerPoint. One slide per scene, black background. Don't add titles — just the image.
4. **VTT tokens:** Crop the 1:1 shots to circles in Roll20/Foundry. Cultist token ×3 (with seed variations), Bone Devil ×1, Vix ×1, Dumal ×1, Moonkite ×1.
5. **DM screen prints:** Vix + Dumal portrait + Senna (face down) clipped to the inside of your DM screen.

## VARIATION STRATEGY (if a gen misses)

- Wrong tone (too clean, too heroic): add `--chaos 30 --weird 50`
- Wrong color (MJ defaults to romantic gold): add `desaturated, drained palette, no warmth`
- Wrong character (looks generic): add reference image of similar character at `--iw 1.0`
- Wrong scale (token too zoomed in): change `--ar` to `1:1` and add `full body shot, head-to-feet visible`
