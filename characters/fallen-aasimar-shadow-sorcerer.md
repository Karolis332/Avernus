**Build target:** Tier 3–4 controller / burst caster with darkness-based battlefield denial.
**Rules:** 2014 PHB + Volo's Guide (Fallen Aasimar) + Xanathar's Guide (Shadow Magic) + TCoE optional rules.

## Signature Combos

1. **Darkness + Sickening Radiance** (lvl 9+) — cast Darkness on a coin, drop it inside Sickening Radiance area. Enemies inside take 4d6 radiant per turn, exhaustion stacks, and have disadvantage on attacks. You see through Darkness; they don't.
2. **Hound + Twinned Polymorph** (lvl 10+) — Hound imposes disadvantage on the WIS save → Polymorph almost certainly lands → Twin it to lock two bosses as turtles.
3. **Subtle Counterspell** — enemy caster has no idea who's blocking them.
4. **Quickened Hypnotic Pattern + Misty Step** — drop the pattern, reposition out of melee in one turn.
5. **Necrotic Shroud + Hound + Disintegrate** — burst nova: +CHA necrotic on Disintegrate, disadvantage save via Hound, ~75 average damage.
